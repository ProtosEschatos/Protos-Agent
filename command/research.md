# /research

Trigger: Assistant agent

Researches a topic using:
- DuckDuckGo Search for current information
- Context7 for library/framework documentation
- Web Fetch for deep page analysis
- GitHub Search for code examples

Produces a structured research summary with key findings, sources, and actionable recommendations.

Usage: `/research <query>`
Examples: `/research best practices for Next.js API routes`
          `/research compare Redis vs Kafka for event streaming`
