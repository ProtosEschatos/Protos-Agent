# /design

Trigger: Frontend Engineer agent

Iterative visual design workflow:
1. Loads the `design-system` skill for CSS best practices, component patterns, and accessibility guidelines
2. Generates layout → styles → animations → responsive polish iteratively
3. Uses Polygram canvas for visual feedback and live iteration
4. Verifies WCAG 2.1 AA compliance (contrast, keyboard nav, semantic HTML)
5. Outputs complete HTML/CSS or React/Vue component

Usage: `/design <description>`
Examples:
- `/design hero section with CTA button and gradient background`
- `/design landing page for fintech startup — dark theme, modern`
- `/design product card grid with hover effects — Tailwind`
- `/design signup form with validation states`
- `/design dashboard sidebar with collapsible sections`
